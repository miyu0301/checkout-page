## How to Use
At first, you need to install Git and Node.js on your computer.  
And execute the following commands.

    # Clone this repository
    $ git clone https://github.com/miyu0301/checkout-page.git

    # Install dependencies
    $ npm install

    # Run the app
    $ npm start
